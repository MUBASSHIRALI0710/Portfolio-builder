# 📁 Portfolio Builder for Students

A powerful, browser-based drag-and-drop tool that helps students create stunning professional portfolios without writing a single line of code. Built with vanilla HTML, CSS, and JavaScript – no frameworks, no dependencies.

![Portfolio Builder Demo](link-to-your-demo-gif-or-screenshot)

## ✨ Features

### Core Functionality
- **🖱️ Drag-and-Drop Interface** – Intuitively drag components from the library onto your canvas
- **📝 Inline Editing** – Click any text to edit directly on the canvas
- **💾 Local Storage** – Auto-save and manual save/load from browser storage
- **🧩 Component Library** – Headers, paragraphs, images with placeholder support
- **🗂️ Template System** – Start with pre-built templates (Minimal, Modern, Classic)

### Advanced Features (Coming in Days 2-3)
- **🎨 Property Panel** – Fine-tune colors, alignment, spacing
- **↩️ Undo/Redo** – Full history navigation with keyboard shortcuts
- **📱 Responsive Preview** – See how your portfolio looks on mobile, tablet, desktop
- **📤 Export** – Download as standalone HTML/CSS file
- **🖼️ Image Upload** – Use your own images instead of placeholders
- **⌨️ Keyboard Shortcuts** – Delete, duplicate, reorder with hotkeys

## 🎯 Why This Project?

This portfolio builder solves a real problem: students need professional portfolios but often lack coding skills or find existing tools too expensive/complicated. With this tool, anyone can:

- Create a portfolio in under 5 minutes
- Customize every aspect visually
- Export a clean, production-ready HTML file
- Host it anywhere (GitHub Pages, Netlify, personal server)

## 🛠️ Built With
- **HTML5** – Semantic structure
- **CSS3** – Flexbox/Grid, modern styling, responsive design
- **Vanilla JavaScript** – No frameworks, pure DOM manipulation
- **LocalStorage API** – Client-side persistence
- **Font Awesome** – Icons for visual appeal
